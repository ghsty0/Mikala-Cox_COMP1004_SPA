CATSWEEPER
This repository contains all necessary code files for the COMP1004 Project, 'CatSweeper'
Stand-Up meeting leader: Mohammad Asad

Project Overview:
CatSweeper is a themed rendition of the logical puzzle game Minesweeper.
Created to aid children's development in problem solving skills as well as improve computer literacy through core game mechanics.
While mostly aimed towards younger audiences, harder difficulties still appeal to older players of the game.

Gameplay & Features:
The goal is to open all non-cat tiles on the board without clicking any cats. If you do, the game ends.
Highscore can be downloaded and saved, then later uploaded to be beaten.
Flagging mechanic aids the user in finding mine locations.
CatSweeper is endlessly replayable due to the randomised gameboard with 3 levels of difficulty.
